Resumo

O objetivo deste trabalho ́e apresentar um sistema composto de dispositivos com atuadores, sensores e um SMA embarcado que realizam controle autônomo de um cultivo. Além disso, este sistema permite conectar os dispositivos em um rede IoT, e com isso, os dados do cultivo são disponibilizados em tempo real. Por fim, uma página web irá requisitar os dados vindos da rede IoT e exibir ao usúario.
